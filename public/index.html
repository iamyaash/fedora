Customizing the Logo The process of customizing the logo is pretty much simple and straightforward. The logos are stored inside assets/ directory. In order to change the logo, Replace the logo.svg with the custom logo (do the same for changing the favicon.svg as well). Run the command in root directory of forgejo/*: make generate-images This process will generate the assets for logo inside the assets/* directory.
